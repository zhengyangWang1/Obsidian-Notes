### 安装配置
[2023 年 MySQL 8.0 安装配置 最简易（保姆级）\_mysql8.0安装配置教程\_mobeicanyue的博客-CSDN博客](https://blog.csdn.net/m0_52559040/article/details/121843945)
安装时需要输入一个密码作为mysql的root权限密码，需要牢记

### 开始使用
在Power Shell中输入：
```
mysql -u root -p
```
然后输入密码进入mysql
