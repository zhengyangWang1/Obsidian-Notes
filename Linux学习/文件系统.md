| 符号  | 含义         |
| ----- | ------------ |
| /     | 根目录       |
| /bin  | 可执行文件   |
| /home | 用户的主目录 |

**特殊目录符号：**

| 目录         |                        释义| 
| ------------ | ---------------------- |
| .            | 代表当前目录           |
| ..           | 代表上层目录           |
| -            | 代表上一个工作目录     |
| ~            | 代表当前用户的主目录   |
| ~accountname | 代表该name账户的主目录 |
